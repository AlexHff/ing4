Recurrent Neural Network (RNN) / Feedback Neural Network (FBNN)

Motivation

With a static dataset, where the data does not change, we can use a FFNN

With a dynamic dataset, the data depends on the past and the future. It is recommended to use a recurrent neural network

Basic RNNs

  • Elman RNN
  • Jordan RNN

There are now many problems that cannot be solved with these networks, but these are still good for learning as they are rather simple

More complex neural networks, that can deal with explosion and disappearing data include

  • Long Short-Term Memory (LSTM)
  • Gated Recurrent Units (GRU)
