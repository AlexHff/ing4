Discriminative Models (DM) and Generative Models (GM)

we characterized machine learning as SL, UL or RL,
but now we can add DM andGM which are rather different too

DM (Discriminative Models)

P(y∣X)P(y | X)P(y∣X)

  1. Represent the dependence of unobserved variables (yyy) on the observed ones (XiX_iXi​)
  2. Infer outputs (yyy) based on inputs (XXX) given some hidden parameters (θ\thetaθ).
  3. Do not care about how the data was generated

GM (Generative Models)

P(X,Y)P(X, Y)P(X,Y)

  1. Generate all values of both those that can be observed (XXX) and those that can be estimated from the observed ones (yyy)
  2. Have as goal:
    1. To model data directly
    2. To estimate the conditional probability density function using the Bayes rule
  3. Estimate how the data was generated

Bayes rule:
P(Y∣X)P(X)=P(X,Y)=P(X∣Y)P(Y)P(Y|X)P(X) = P(X, Y)=P(X|Y)P(Y)P(Y∣X)P(X)=P(X,Y)=P(X∣Y)P(Y)
P(Y∣X)=P(X,Y)P(X)=P(X∣Y)P(Y)P(X)P(Y|X)=\frac{P(X, Y)}{P(X)}=\frac{P(X|Y)P(Y)}{P(X)}P(Y∣X)=P(X)P(X,Y)​=P(X)P(X∣Y)P(Y)​
